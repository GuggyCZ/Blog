Nová verze IIS Network Diagnostic Tools

 24. srpna 2010
  •  Software

Dnes jsem na CodePlexu zveřejnil novou verzi 1.1 svého nástroje IIS Network Diagnostic Tools. Tato verze opravuje jednu potenciální bezpečnostní chybu a přidává žádanou funkci resolvingu IP adres na host names při traceroute.

Nejnovější verzi si můžete vždy stáhnout ze stránky http://iisnetdiag.codeplex.com/releases/

Podrobnější informace o funkcích a určení IIS NetDiag a příkazů ping a traceroute najdete v mém starším článku IIS Network Diagnostic Tools: Ping a Traceroute pro váš server.
