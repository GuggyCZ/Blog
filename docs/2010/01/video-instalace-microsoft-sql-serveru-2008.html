Video: Instalace Microsoft SQL Serveru 2008

Dnes vychází na MSTV.CZ třetí díl videoprůvodce instalací web serveru na Windows 2008. Pokud to s provozem webových aplikací myslíte alespoň trochu vážně, bez databází se neobejdete. Ukážeme si, jak můžete nainstalovat Microsofft SQL Server 2008 – buďto Express edici, která je dostupná zdarma, nebo komerční Web edici, která je přímo pro web určena.
