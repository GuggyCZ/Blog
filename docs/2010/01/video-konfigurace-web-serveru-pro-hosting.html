Video: Konfigurace web serveru pro hosting

Trvalo to pouhých osm dílů a v našem seriálu na MSTV.CZ se dostáváme konečně k vlastní konfiguraci web serveru. Podíváme se na správné nastavení web serveru, při kterém budou jednotlivé aplikace, resp. jednotliví zákazníci od sebe bezpečně izolováni. Zaměříme se na správné nastavení application poolů, práv ve file systému a související záludnosti.
