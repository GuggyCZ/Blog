Video: Údržba databází na SQL Serveru

 20. ledna 2010
  •  IIS

Je zde středa a MSTV.CZ přináší další video o instalaci web serveru. Ačkoliv vám principiálně nic nebrání minule popisovanou metodu záloh používat i u vyšších edic, než jenom Express, není to nutné. Vyšší edice SQL Serveru než Express mají SQL Agent, který se o zálohování a údržbu databází postará. Ukážeme si, jak jej nakonfigurovat a jak si s ním povídat pomocí e-mailových zpráv.
